Summary

The Listen dataset is a dataset of electrocardiogram (ECG) and pulse oximetry (photoplethysmogram, PPG) signals acquired from patients in hospital. This dataset is intended to facilitate evaluation of the performance of algorithms in a range of clinical settings.

The dataset contains recordings from patients whilst recovering from cardiac surgery.

Work is still ongoing on this dataset.
