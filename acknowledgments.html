Additional Contributions

Some components of the toolbox of algorithms use additional open source components. Specifically, the QRS detector used was written by Prof. Gari Clifford, and is available here . In addition, the Principal Component Analysis technique uses code from the LS-SVMlab toolbox, written by the group at K.U. Leuven university.
